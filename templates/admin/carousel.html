How it works
  • Slide 0 is the Welcome/Brand slide — it never changes.
  • Slides 1–4 each show one book. Pick any active book from the dropdown.
  • If you leave a slide on "keyword fallback", it auto-picks the first book matching the original keyword (Bhagavad Gita, Bhagavatam, Krishna, Nectar).
  • The "Shop Now" button on each slide links directly to that book's detail page.
  • For best display, upload a cover image for the book via Books → Edit.
